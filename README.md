Guiping is a Gui(Graphical User Interface)Ping Application Tool.

### Setup[PIP MODULE] ###

+ sudo apt install python3-tk
+ sudo pip3 install -r requirements.txt

### Terminal Command ###

+ git clone https://github.com/r3k4t/guiping.git
+ cd guiping
+ sudo python3 guiping.py
